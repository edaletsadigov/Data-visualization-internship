## 🗂️ Project Overview

Analysis of Telco Customer Churn dataset to identify churn patterns 
and visualize customer behavior across multiple dimensions. 
All 5 tasks completed in a single Excel workbook — each task on a separate sheet.

---

## 📁 Workbook Structure

| Sheet | Task | Description | Skills |
|-------|------|-------------|--------|
| TASK 1 | Data Overview & Simple Sorting | Explored dataset structure, sorted by Tenure, filtered churned customers | Sorting, Filtering |
| TASK 2 | Churn Count Visualization | Column chart showing churned vs. non-churned counts with labels | Column Charts |
| TASK 3 | Monthly Charges Distribution | Histogram of MonthlyCharges with bin size = 10 | Histograms |
| TASK 4 | Churn Rates by Gender | Pivot Table + Clustered Bar Chart by gender | Pivot Tables, Bar Charts |
| TASK 5 | Heatmap — Monthly Charges & Tenure | Conditional formatting heatmap: MonthlyCharges × Tenure × Churn | Heatmap, Data Interpretation |

---

## 📌 Dataset

**Telco Customer Churn**  
- 7,043 customer records  
- Features: Gender, Tenure, MonthlyCharges, TotalCharges, Churn, and more  

---

## 🔑 Key Findings

- Churned customers represent a measurable minority — visible in Task 2 column chart
- Monthly charges distribution shows clear concentration in mid-to-high range (Task 3)
- Gender does not appear to be a significant churn driver (Task 4)
- High monthly charges + low tenure = highest churn risk segment (Task 5 heatmap)
